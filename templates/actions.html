How to Connect with Key
After downloading the .pem file:
  1. Set correct permissions on your local machine:
    chmod 400 username-hybridsec.pem
  2. Connect using SSH:
    ssh -i username-hybridsec.pem user@<server-ip>
  3. On Windows (PuTTY): Convert .pem → .ppk using PuTTYgen,
    then load the .ppk in PuTTY → Connection → SSH → Auth.
  4. Once key login works, disable password auth in SSH Settings above for maximum security.
